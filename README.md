🔔Actualización del Repositorio: Algoritmo de las Torres de Hanói
Este repositorio ha sido actualizado con la implementación del algoritmo clásico de las Torres de Hanói, un excelente ejemplo de la aplicación de la recursión en la informática.


🧩Implementación 
El archivo TorreHanoi.py contiene la solución algorítmica óptima para el problema de las Torres de Hanói.


🔑Concepto Central: Recursión
La solución se basa completamente en una función recursiva. La recursión es una técnica donde una función se llama a sí misma para resolver un problema más pequeño hasta alcanzar un caso base que detiene el proceso.


🎯En el contexto de Hanói, la función descompone el problema de mover n discos en tres pasos:

Mover los n−1 discos superiores a la varilla auxiliar.

Mover el disco n (el más grande) al destino final.

Mover los n−1 discos de la varilla auxiliar al destino final.
